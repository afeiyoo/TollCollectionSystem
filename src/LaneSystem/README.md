# TollCollectionSystem
收费软件系统

## 工程目录结构

```bash
TollCollectionSystem
├── 3rdparty      # 第三方库的存储目录
│   ├── CuteLogger
│   ├── EasyQtSql
│   ├── ElaWidgetTools
│   ├── optional
│   └── utils
├── bin          # 可执行文件存储目录
├── build        # 构建产物存储目录
├── docs         # 相关项目文档存储目录
├── lib          # 库文件存储目录
├── src          # 源码存储目录
│   ├── devices          # 设备相关项目存储目录
│   ├── mainsys          # 车道收费系统主项目存储目录
│   │   ├── bend         # 后端
│   │   ├── config       # 配置
│   │   ├── core         # 核心，后端服务
│   │   ├── dbs          # 数据库相关
│   │   ├── global       # 全局对象
│   │   ├── gui          # 前端
│   │   ├── middle       # 中间层
│   └── tools            # 相关Demo项目
│       └── InfobardManager
├── static               # 资源文件存储目录
│   ├── config
│   └── images
└── tests                # 测试代码
```
